# Keyboard Shortcuts for Task Management

This document outlines the keyboard shortcuts available in the task management interface.

## Global Shortcuts (available on the tasks page)

| Shortcut | Action |
|----------|--------|
| `Ctrl + N` | Focus on the new task input field |

## Task-Specific Shortcuts (available when a task item is focused)

| Shortcut | Action |
|----------|--------|
| `Space` | Toggle task completion status |
| `Delete` | Delete the task |
| `Ctrl + E` | Edit the task |
| `Ctrl + S` | Save changes when editing |
| `Escape` | Cancel editing |

## How to Use

1. **Global shortcuts** work anywhere on the tasks page
2. **Task-specific shortcuts** require focusing on a specific task item first (click on the task or tab to it)
3. All shortcuts are designed to be intuitive and consistent with common applications

## Accessibility Notes

- All keyboard shortcuts have visual alternatives via mouse/touch interactions
- Shortcuts are documented in this guide for discoverability
- Users can rely solely on keyboard navigation for all task operations
- Focus indicators are visible to show which element has keyboard focus
